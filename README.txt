Проект реализует веб-приложение для управления контактами.
Бэкенд написан на ASP.NET 7, фронтенд — на чистом JavaScript, без использования фреймворков (имею опыт работы с React и Angular).

Функциональность включает:
Просмотр списка контактов
Создание, редактирование, удаление контактов
Валидация полей на стороне клиента
Использование всплывающих окон (модальных окон) для добавления/редактирования


Для запуска Frontend части:
Открой папку проекта в VS Code.
В терминале введи:
npm install
npm run dev


Для запуска Backend части:
Перейдите в директорию TestTask.API
Выполните команду:
dotnet run
